Cell Maps for Artificial Intelligence

Version 1.0 License ↗ Released 2026-07-07

Datasheet Summary

The CM4AI data generation project seeks to map the spatiotemporal architecture of human cells and use these maps toward the grand challenge of interpretable genotype-phenotype learning. It computes hierarchical cell maps using AI/ML methods, on protein interaction and localization data, with perturb seq data, for disease relevant cell lines, and provides data in AI-ready form.
